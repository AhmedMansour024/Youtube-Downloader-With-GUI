# Youtube Downloader With GUI

GUi Program Made with Python using yt_dlp library.

## Description

Easy to use Youtube Downloader that can download many videos

## How to run ?

make sure to run **make** :

```bash
make
```

then you can run **main.py** by running :

```bash
python main.py
```

in the Terminal.
